# ブロック崩し改
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
## ゲームの概要
ブロック崩しをもとにいくつかの機能を追加

## ゲームの実装
### 共通基本機能
* ブロックを描画
### 担当追加機能
* ボールのクラス
    - ボールを描画して、動かす
    - 画面の下部に接触した場合、ボールは消える
* 接触判定に関する関数
    - ボールとオブジェクトが接触したときに、ボールのどこが接触したか判定する
* オブジェクトが画面外へ出たかどうか判定する関数
    - 今までの授業で使ったcheck_boundと同じもの、ただし名前をcheck_bound_outに変えている
- 背景を追加
